# Next.js

- [公式サイト](https://expressjs.com/ja/)

## Next.js とは

React のフレームワークで、SSR（サーバーサイドレンダリング）や静的サイト生成(SSG)を簡単に行えます。SEO の最適化や、高速なページ遷移が求められる Web アプリケーションの開発に適しています。

# 構成と機能

ファイルベースのルーティングがデフォルトで組み込まれており、ディレクトリ構造に従って自動でルートが作成されます。また、ページ単位で設定できるビルド最適化やコード分割機能もあり、使いやすく設計されています。

## 何に使われるのか

- フロントエンド主導のアプリケーションや Web サイトに最適です。特に SEO が重要な Web アプリや、動的に生成されるページが多いプロジェクトに向いています。
