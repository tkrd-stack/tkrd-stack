# JavaScript

## 概要

JavaScript は、Web 開発において最も広く使われるプログラミング言語の 1 つです。

## 学習ポイント

- 変数・データ型
- 関数・スコープ
- 非同期処理（Promise, async/await）
- DOM 操作
- ES6 以降の新機能

## コードスニペット

```js
console.log('Hello, JavaScript!');
```
